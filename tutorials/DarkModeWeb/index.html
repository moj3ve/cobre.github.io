How To Integrate MacOS Mojave's Dark Mode Into Your Website

In MacOS 10.14.4, Apple added dark mode support for websites. It's pretty easy to enable for your website. All you have to do is add a couple lines of css:

@media (prefers-color-scheme: dark) {

body {
color: #ffffff;
background-color: #232224;
}

}

Of course you can change the colors, and you should probably add more css to it. But there you go! You should now be able to change from dark mode to light mode and have your website change with it!
