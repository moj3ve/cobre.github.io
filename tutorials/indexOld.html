How To Create A Cydia Repo With GitHub

Getting Started

What You Need:

Computer
- I am using a Mac, but this should work on any other OS
.deb
- This is the compiled package of the tweak you want to upload
iOS Device To Test Things Out On
- This is optional

1. Starting Out

First, go to Github.com and create a new repository. If you want it to be yourusername.github.io, name the repository yourusername.

Example: For me, it would be called mac-user669 and I would add mac-user669.github.io to Cydia.

If you would like it to be yourusername.github.io/repo (or whatever else), name the repository repo or whatever you decide to append to the end.

Example: For me, it would be called repo and I would add mac-user669.github.io/repo to Cydia.

If you wish, you can add a README, but you are welcome to copy the one from this tutorial. Download a tutorial repo to you computer with cd Desktop (It doesn't have to be on the Desktop, but to make it easy, it's going to be there) and thengit clone https://github.com/supermamon/Reposi3.git. Clone your newly created repo by copying the link given when you click "Clone or Download". Open the newly created folders, and copy the contents of Reposi3 into your repo. Open up the Release file, and change anything that says Reposi3 to your username/screen name. Do the same in index.html. If you're on a Mac, you should open this with Atom, which is a really fancy text editor, among other things. Delete the two sample packages from the debs folder, and input your own. Rename depictions/com.supermamon.newpackage to the Bundle ID of your package, and modify changelog.xml, info.xml, and add your screenshots to screenshots. Once you're done modifying and adding your stuff, open Terminal, and run these commands: cd Desktop/NameOfYourRepo dpkg-scanpackages -m . /dev/null >Packages && bzip2 -zk Packages git add -A git commit -m "Whatever you changed" git push origin master

2. Enabling GitHub Pages

Open your new repo at github.com, and go into the settings. Scroll down to the section about GitHub Pages, and select the master branch. If everything worked, it should have said "Your site is published at" and then a URL. Open that URL, and make sure everything displays correctly. Finally, add it to Cydia! You're done!

3. Troubleshooting

Sooner or later, something will go wrong. Check here to see how to fix it! I'll add more over time.

I'm getting a Hash Mismatch error in Cydia
Rerun the last set of commands from the second part.
